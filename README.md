# EV3-CrabRave-Game
###### Lavet i samarbejde med:
- Frederik Friis
- Christian Schibler
- Mads Brogaard

## Beskrivelse
Vi laver et spil som på baggrund af vores viden af Googles Dinosaur spil. Spillet forgår på displayet og man skal bruge en tryksensor som er knappen til at hoppe. 

### Nødvendigheder
- Tryksensor (x1)
- EV3 Blok (x1)
